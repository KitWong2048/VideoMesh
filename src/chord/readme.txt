This is a project aims at implement a distributed hash table system , Chord. Source files and documentation can be found in the "src" and "doc" folders respectively. Please refer to the file "DistributedTest.cpp" in the folder "src/Test" for how to use the system. In short, application can interface the system via the classes "ChordApp" and "AsyncHandle", which can be found in the folder "src/App". To compile the source files, type "make" under the "src".

Future extensions include but not limited to the followings,
	- add a recursive function call
	- handle private IP issue
